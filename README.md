# 🐍 Console Snake Game (C++)

A classic Snake Game built entirely in **C++** for the Windows console. This project was created to strengthen my understanding of arrays, game loops, functions, collision detection, and basic game development concepts.

---

## 🎮 Features

- Move the snake using **W A S D**
- Random food generation
- Snake grows after eating food
- Score counter
- Wall collision detection
- Self-collision detection
- Prevents reversing directly into the snake
- Food never spawns on the snake's body
- Simple console-based graphics

---

## 🛠 Technologies Used

- C++
- Windows Console
- `<conio.h>` for keyboard input
- `<thread>` and `<chrono>` for game timing
- `<cstdlib>` and `<ctime>` for random food generation

## ▶ How to Run

### Compile

```bash
g++ Snake.cpp -o snakecpp.exe
```

### Run

```bash
./snakecpp.exe
```

or on Windows PowerShell

```powershell
.\snakecpp.exe
```

---

## 🎮 Controls

| Key | Action |
|------|--------|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |
| Q | Quit Game |
